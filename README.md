## 📌 Project Description

The Blood Bank Management System is a web-based application developed to efficiently manage blood donors, blood groups, and blood availability. It helps in maintaining records in an organized way and allows quick access during emergencies.

---

## 🚀 Features

✔ Add and manage donor details
✔ Track blood groups and availability
✔ Search for required blood types
✔ Simple and user-friendly interface
